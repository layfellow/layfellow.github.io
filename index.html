Projects

Both personal and collaborative
cPanel CLI A command line interface for the cPanel Unrestricted API (UAPI).

Quickly execute commands in a terminal to manage a website instead of logging into cPanel and clicking your way around the web interface. You can also use this utility in scripting jobs.
cPanel API Forked from tz4678/cpanel-api. cPanel 2 API Client for Python.

I added an option to authenticate using UAPI tokens. I use it for cPanel CLI.
IMAP Copy Forked from r00tat/imap_copy. A simple utility to copy folders between IMAP mail servers.

Forked to add an option to test the IMAP connection and list remote folders. Also, fixed a couple of bugs I found for Hotmail and Gmail. I actually used this to migrate a decade’s worth of email content when switched email providers.
Auctores This is a pointless art project. (No one said that art had to make sense.)

It’s an exercise in wasting good code for nothing.
