Projects

Both personal and collaborative
IMAP Copy Forked from r00tat/imap_copy. A simple tool to copy email folders from one IMAP server to another.

I forked it to add a couple of options to test the IMAP connection and list remote folders. I actually used this to migrate a decade’s worth of email content when I switched email providers.
cPanel API Forked from tz4678/cpanel-api. cPanel 2 API Client for Python.

I added an option to authenticate using UAPI tokens. Currently building a full blown cPanel 2 CLI utility on top of it.
Auctores This is a pointless art project. (No one said that art had to make sense.)

It’s an exercise in wasting good code for nothing.
