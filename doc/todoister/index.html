Todoister

Todoister is a simple Todoist CLI client written in Go.

Commands

  • version - Print the version number
  • list - List projects
  • tasks - List project tasks
  • export - Export projects in JSON or YAML format

version

todoister version [flags]

Print the Todoister version number.

Global Options

-t, --token <string>
use <string> as Todoist API token

list

todoister list [project]... [flags]

List projects and subprojects.

project is the name of one or more projects to list tasks from. If no project is given, all projects are listed. You can specify a project name by its full path, e.g., Work/Project. Names are case-insensitive.

Global Options

-t, --token <string>
use <string> as Todoist API token

Examples

# List all projects and subprojects:
todoister ls

# List projects Work and Life and their subprojects:
todoister ls Work Life

# List all subprojects of Project, which is a subproject of Work:
todoister ls Work/Project

tasks

todoister tasks project... [flags]

List project tasks.

project is the name of one or more projects to list tasks from. You can specify a project name by its full path, e.g., Work/Project. Names are case-insensitive.

Global Options

-t, --token <string>
use <string> as Todoist API token

Examples

# List tasks for project Life:
todoister tasks Life

# List tasks for subproject Project of project Work:
todoister tasks Work/Project

export

todoister export [path] [flags]

Export all Todoist projects as a tree of JSON or YAML files.

  • path is a file or directory where to export the projects, by default index.json.

Options

-d, --depth <int>
depth of subdirectory tree to create on the filesystem when exporting (default is 0, i.e., no subdirectories)
--json
export in JSON format (default)
--yaml
export in YAML format

Global Options

-t, --token <string>
use <string> as Todoist API token

Examples

# Export to a single index.json file in the current directory:
todoister export

# Export to todoist.json file in the home directory:
todoister export ~/todoist.json

# Export to todoist.yaml file in the home directory:
todoister export ~/todoist.yaml --yaml

# Export to a projects directory in the home, with subdirectories down to 3 levels deep:
todoister export ~/projects --json -d 3
